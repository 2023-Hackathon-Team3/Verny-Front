# Verny_Front


- **서비스 소개**


**모두가 문화예술을 각자의 감각으로 받아들일 수 있도록, 베르니 (Verny)**

스크린리더를 사용하는 시각장애인 사용자 역시 불편함 없이 사용 가능한 배리어프리 문화예술 플랫폼, 베르니 (Verny) 입니다.

음성해설문 가이드라인을 준수한 고전·현대미술품의 기술 및 해석과, 배리어프리 문화예술관광지 정보를 제공합니다.
